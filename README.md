# How To Setup
1. Clone the T3 repo - https://gitenterprise.inside-box.net/Box/T3/
2. git checkout gh-pages
3. Install jekyll `gem install jekyll`
4. `jekyll serve --watch`
5. Open localhost:4000/pages/Box/T3/
